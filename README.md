## Description
This is a simple Drawing Pad application developed using Python and Tkinter. It allows users to draw on a canvas using different colors and brush sizes.

## Features
- Draw using the mouse
- Change drawing colors
- Choose custom colors
- Adjust brush size
- Eraser option
- Clear the canvas

## Technologies Used
- Python 3.x
- Tkinter

## How to Run the Project
1. Install Python.
2. Download the project files.
3. Open the project folder.
4. Run the following command:

```bash
python main.py
```
